# 我的库表与数据专题优化

## 一、我的库表 UI 优化

### 1. 页面标题优化 — [MyTables/index.vue](datapilot-datamap/src/pages/MyTables/index.vue)

当前 `PageHeader title="我的库表"` 使用默认样式，效果较单调。优化方向：
- 给 PageHeader 增加描述文案（如"管理您负责、收藏和最近访问的数据表"），利用 `extra` slot 或在 `PageHeader` 下方增加 subtitle
- 调整标题样式，去掉底部分割线或使其与 tabs 融合，使标题区域更紧凑美观

### 2. 表格列优化 — [MyTables/TableGroup.vue](datapilot-datamap/src/pages/MyTables/TableGroup.vue)

**数据源列**（第 26-28 行）：当前 `<SourceTag>` 只展示 icon，需改为 `icon + "Hive"` 文字，与资产检索列表页一致：
```vue
<div class="flex items-center gap-1.5">
  <SourceTag :type="record.serviceType" :size="16" />
  <span class="text-[13px]">{{ record.serviceType }}</span>
</div>
```

**分层列**（第 29-31 行）：当前 `<a-tag>` 无 `color` 属性，需添加 `color="blue"` 与列表页/详情页保持一致：
```vue
<a-tag color="blue">{{ record.layer }}</a-tag>
```

**描述列**（第 80 行）：当前 `ellipsis: true` 只做表格级省略，缺少 tooltip。需在 bodyCell 中增加描述列的渲染模板，使用 `<a-tooltip>` 包裹

**列顺序调整**：将 `columns` 数组中"热度"列移到"更新时间"前面，并将两个热度指标改为**水平一行**展示（改 `flex-col` 为 `flex-row`），让布局更紧凑：
```
表名 | 数据源 | 分层 | 描述 | 热度(使用+浏览) | 更新时间
```

### 3. 热度一行展示 — [MyTables/TableGroup.vue](datapilot-datamap/src/pages/MyTables/TableGroup.vue)

将当前 usage 列中的两个热度指标从 `flex flex-col` 改为 `flex flex-row gap-3`，使 "使用热度" 和 "浏览热度" 在同一行展示。

## 二、数据专题优化

### 4. 增加搜索功能 — [Topics/index.vue](datapilot-datamap/src/pages/Topics/index.vue)

在页面标题下方、卡片网格上方增加搜索栏：
- 使用 `a-input-search` 组件，placeholder="搜索数据专题"
- 搜索逻辑：对 `topics` 数组进行前端过滤（匹配 `name`、`description`、`domain`）
- 搜索结果为空时展示 `a-empty`

### 5. 新建专题入口 — [Topics/index.vue](datapilot-datamap/src/pages/Topics/index.vue)

- 在搜索栏右侧增加"新建专题"按钮（`a-button type="primary"`，带 `PlusOutlined` 图标）
- 点击后弹出 `a-modal`，展示新建专题表单

### 6. 新建/编辑专题弹窗（Modal）— [Topics/index.vue](datapilot-datamap/src/pages/Topics/index.vue)

参考 DataLeap Asset Catalog/Album 的设计，表单包含以下字段：
- **专题名称**：`a-input`，必填
- **专题描述**：`a-textarea`，选填
- **专题标签**：`a-select mode="tags"`，支持自定义输入标签
- **所属业务域**：`a-select`，选项从 domainColorMap 中获取（交易域/用户域/营销域/风控域/内容域/基础域）
- **专题负责人**：`a-select mode="multiple" showSearch`，支持多选和关键词搜索（mock 负责人列表：张三、李四、王五等）
- **是否首页展示**：`a-switch`，默认关闭，控制是否在首页展示
- **添加数据表**：使用 `a-cascader` (级联选择) 或自定义的多级选择器，实现层级关系：数据源 (Hive/MySQL) -> 库 (dwa_cn/dm_cn) -> 表 (具体表名)，支持多选。

表单底部："取消"和"保存"按钮。保存后将新专题追加到 topics 列表中或更新已有专题（前端 mock 操作，`message.success` 提示成功）。

### 7. 专题编辑功能 — [Topics/TopicCard.vue](datapilot-datamap/src/pages/Topics/TopicCard.vue)

在每个专题卡片上（例如右上角或 hover 时显示），增加一个"编辑"按钮（`EditOutlined` 图标）。点击后触发父组件的编辑事件，打开与新建相同的 Modal 弹窗，并回填当前专题的数据。

### 8. 专题 mock 数据中移除"周八" — [Topics/index.vue](datapilot-datamap/src/pages/Topics/index.vue)

第 41 行 `owner: '周八(zhouba)'` 需要替换为其他用户。
