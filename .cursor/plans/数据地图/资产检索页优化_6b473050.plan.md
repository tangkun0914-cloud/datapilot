# 资产检索页优化计划

## 涉及文件

- [SearchBar.vue](src/pages/Search/SearchBar.vue) -- 搜索框组件
- [index.vue (Search)](src/pages/Search/index.vue) -- 搜索页主容器
- [FilterPanel.vue](src/pages/Search/FilterPanel.vue) -- 左侧筛选面板
- [ResultList.vue](src/pages/Search/ResultList.vue) -- 列表模式卡片
- [ResultTable.vue](src/pages/Search/ResultTable.vue) -- Excel 模式表格

---

## 1. 搜索框：搜索按钮内嵌 + 收藏按钮缩小

**文件**: `SearchBar.vue`

- 将"搜索"按钮从输入框外部移入 `a-input` 的 `suffix` 插槽内，使其视觉上嵌入输入框右侧
- 输入框改为带圆角的一体化搜索框，搜索按钮紧贴输入框右侧边缘
- "我的收藏"按钮改为 `size="middle"` 或 `size="small"`，缩小整体体积

## 2. 搜索结果提示文案：仅搜索词存在时展示

**文件**: `Search/index.vue`

- `toolbar-left` 区域增加 `v-if` 条件判断：当 `keyword.trim()` 不为空时才展示"为您找到 X 个搜索结果"文案
- 无搜索词时（首页直接跳转、查看更多），该区域不展示

## 3. 排序下拉：移除"综合排序"，默认更新时间

**文件**: `Search/index.vue`

- 移除 `<a-select-option value="relevance">综合排序</a-select-option>`
- 仅保留两个选项：更新时间（`updatedAt`）、热度（`usage`）
- `sortBy` 默认值从 `'relevance'` 改为 `'updatedAt'`
- 对应 `onMounted` 和 `watch` 中对 `sort` 参数的处理逻辑同步调整

## 4. Excel 模式工具栏 UI 优化（位置不变，仅改样式）

**文件**: `ResultTable.vue`

- 三个按钮（导出、复制、列显示）保持在表格上方独立一行，位置不动
- 优化按钮样式：改为统一的灰色描边风格，`size="small"`，字体 13px，图标颜色 `#64748b`
- 按钮 hover 时边框和文字变蓝（`#1677ff`），增加过渡动画
- `.table-toolbar` 右对齐（`justify-content: flex-end`），按钮间距缩小至 8px
- 按钮文字与图标间距紧凑，整体更精致、不突兀

## 5. Excel 模式更新时间列支持排序

**文件**: `ResultTable.vue`

- 在 `allColumns` 的 `updatedAt` 列增加 `sorter: (a, b) => (a.updatedAt || 0) - (b.updatedAt || 0)` 属性
- 增加 `defaultSortOrder: 'descend'` 默认降序

## 6. 筛选面板：数据库和负责人支持关键词搜索

**文件**: `FilterPanel.vue`

- "库" 的 `a-select` 增加 `showSearch` 和 `filterOption` 属性，允许用户通过输入关键词筛选数据库选项
- "负责人" 的 `a-select` 同样增加 `showSearch` 和 `filterOption`，支持按姓名/拼音搜索
- `filterOption` 实现：对 `option.label` 或 `option.children` 做 `toLowerCase().includes()` 模糊匹配

## 7. 列表模式：表名过长时 `...` 省略 + tooltip

**文件**: `ResultList.vue`

- `.table-name` 增加 `max-width`、`overflow: hidden`、`text-overflow: ellipsis`、`white-space: nowrap` CSS
- 将现有的 `:title` 属性改为 `a-tooltip`，鼠标悬停时展示完整 `database.table_name`

## 8 和 9. 列表模式：数据层级改为"分层"标签 + 所属库标签展示

**文件**: `ResultList.vue`

- "数据层级" label 改为"分层"
- 分层值使用 `<a-tag>` 标签展示（如 `<a-tag color="blue">DWA</a-tag>`），而非纯文本
- "所属库" 值同样使用 `<a-tag>` 标签展示（如 `<a-tag>dwa_cn</a-tag>`），而非纯文本
