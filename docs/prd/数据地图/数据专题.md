# 数据专题

## 一、列表页

### 1.1 筛选项
| 筛选条件 | 说明 |
|---------|------|
| 专题名 | 关键词模糊搜索 |
| 全部 | 展示所有专题 |
| 我管理的专题 | 当前用户为管理员的专题 |
| 我创建的专题 | 当前用户创建的专题 |
| 我关注的专题 | 当前用户已关注的专题 |

### 1.2 专题卡片
以卡片形式展示，每张卡片包含：

- **专题名称**
- **专题描述**
- **负责人**
- **表数量**
- **关注人数**
- **创建时间**
- **最近一次更新时间**
- 操作按钮：`关注` · `编辑` · `删除`

### 1.3 创建专题

点击「创建专题」按钮，弹窗填写：

| 字段 | 说明 |
|------|------|
| 专题名称 | 必填 |
| 可见范围 | 公开 / 私有 |
| 专题描述 | 选填 |

点击确认后，跳转至 **专题维护页**。
